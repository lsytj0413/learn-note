# 第十章: 进程 #

本章介绍的命令如下:

- ps: 显示当前所有进程的运行情况
- top: 实时显示当前所有任务的资源占用情况
- jobs: 列出所有活动作业的状态信息
- bg: 设置在后台中运行作业
- fg: 设置在前台中运行作业
- kill: 发送信号给某个进程
- killall: 杀死指定名字的进程
- shutdown: 关机或者重启系统
