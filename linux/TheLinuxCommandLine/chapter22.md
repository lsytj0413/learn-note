# 第二十二章: 打印 #

本章介绍的命令如下:

- pr: 转换文本文件, 从而进行打印操作
- lpr: 打印文件
- lp: 打印文件(System V)
- a2ps: 格式化文件, 以在PostScript打印机上打印
- lpstat: 显示打印状态信息
- lpq: 显示打印机队列状态
- lprm: 取消打印任务
- cancel: 取消打印任务(System V)
